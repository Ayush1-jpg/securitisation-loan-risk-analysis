# 📊 Securitisation Loan Risk Analysis

## Project Overview

This project analyzes a securitised loan portfolio using Microsoft Power BI. It provides interactive dashboards to monitor portfolio performance, credit risk, IFRS 9 Expected Credit Loss (ECL), and static pool vintage analysis.

The project demonstrates end-to-end Business Intelligence development, including data transformation, data modelling, DAX calculations, and dashboard design.

---

## Business Objectives

- Monitor loan portfolio performance
- Analyze credit risk across the portfolio
- Track portfolio losses and recoveries
- Evaluate IFRS 9 Expected Credit Loss (ECL)
- Perform static pool vintage analysis
- Generate business insights through interactive dashboards

## Dashboards Included

### Executive Dashboard
Provides a high-level overview of the loan portfolio using business KPIs and portfolio distribution visuals.

### Credit Risk Dashboard
Analyzes delinquency buckets, DPD distribution, outstanding balances, and vehicle type risk.

### Portfolio Performance Dashboard
Tracks monthly net losses, recoveries, defaults, and prepayments.

### Vintage Analysis Dashboard
Evaluates pool factor trends, cumulative net loss rate, prepayments, and remaining pool balance across vintages.

### IFRS 9 / ECL Dashboard
Visualizes Expected Credit Loss (ECL), Exposure at Default (EAD), Loss Given Default (LGD), and IFRS Stage analysis.
